# Branding: Merita — App Interna

## Concepto Creativo: "Precisión Silenciosa"

La tecnología debe ser invisible pero impecable. La marca no debe gritar "software médico", sino **"asistente de alta costura"**.

> **Slogan interno:** _"Merita: Excelencia en el detalle, precisión en el cuidado."_

---

## Paleta de Colores: "Neutral Luxury & Functional Coding"

Base neutra para la interfaz general. El color actúa **solo como indicador funcional**.

### Base Neutra (Elegancia)

| Nombre         | Hex       | Uso                                           |
| -------------- | --------- | --------------------------------------------- |
| Midnight Ash   | `#2C2C2C` | Textos principales y barras de navegación     |
| Champagne Mist | `#F5F2ED` | Fondo principal — más cálido que blanco puro  |
| Silver Silk    | `#D9D9D9` | Líneas divisorias y estados inactivos         |

### Acentos Funcionales (Diferenciación de Turnos)

| Turno / Estado    | Nombre          | Hex       | Concepto                              |
| ----------------- | --------------- | --------- | ------------------------------------- |
| Turno Mañana      | Sunrise Ochre   | `#E4A853` | Dorado apagado — energía controlada   |
| Turno Tarde       | Serene Sage     | `#829A84` | Verde musgo — calma                   |
| Turno Noche       | Royal Indigo    | `#3F4B6B` | Azul profundo — vigilancia y silencio |
| Alertas/Accidentes| Terracotta Soft | `#C36A59` | Rojo teja — alerta sin pánico visual  |

## Tipografía: "Formalidad Técnica"

### Títulos — Serif

- **Lora** o **Playfair Display**
- Comunican tradición, respeto y herencia
- Peso recomendado: **Semi-Bold**

### Cuerpo de texto y botones — Sans Serif

- **Montserrat** o **Inter**
- Máxima legibilidad para personal en movimiento

| Peso    | Uso                    |
| ------- | ---------------------- |
| Light   | Datos secundarios      |
| Regular | Cuerpo de texto        |
| Medium  | Nombres de pacientes   |
| Bold    | Botones de acción      |

---

## Moodboard Visual

Interfaz con mucho aire (espacio en blanco). Tarjetas con bordes sutilmente redondeados, sin sombras pesadas — solo elevación mínima. Fotos de pacientes/personal en marcos circulares.

Referente visual: **interfaz de hotel 5 estrellas o app de banca privada** — limpia, costosa, extremadamente organizada.

---

## Tono de Voz

| Atributo     | Descripción                                                                 |
| ------------ | --------------------------------------------------------------------------- |
| Directo      | Sin diminutivos ni lenguaje excesivamente cercano                           |
| Eficiente    | Instrucciones cortas: "Confirmar administración" (no "¿Ya le diste la med?") |
| Responsable  | En sección de accidentes: lenguaje objetivo y clínico                       |

## Estructura UX Sugerida

### Dashboard de Turno
Visualización clara de tareas del día con el código de color del turno correspondiente.

### Bitácora de Incidentes
Botón flotante de acceso rápido (Terracotta `#C36A59`) para registrar accidentes. Incluye adjuntar foto y hora automática.

### Calendario de Reservas
Sistema de drag & drop para que el personal elija sus días, sujeto a aprobación del admin.

### Panel de Control (Admin)
Vista de **mapa de calor** para ver qué tareas se están completando y dónde hay retrasos.
